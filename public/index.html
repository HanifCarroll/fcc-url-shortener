URL Shortener Microservice

This app will convert a long URL to something a bit more manageable.

How to use:

Simply add /new/ to the end of this link, followed by the complete URL (http/https included!), then press enter!

Example input:

https://hmcshort.herokuapp.com/new/https://www.google.com

Example output:

{ "Original URL": "https://www.google.com",
"Shortened URL": "https://hmcshort.herokuapp.com/BJbT3fdLb" }

Voila!

You now have a shortened URL for your personal use! Visiting https://hmcshort.herokuapp.com/BJbT3fdLb will redirect you to https://www.google.com"!
